## 摘要
大语言模型（LLMs）将海量的世界知识内化为参数化记忆，但不可避免地继承了其源语料库的陈旧性和错误。因此，确保这些内部表示的可靠性和可塑性对于可信的现实世界部署至关重要。知识编辑提供了一种关键范式，可以在不进行重新训练的情况下对记忆进行手术式修改。然而，尽管最近的编辑器在标准基准测试中表现出很高的成功率，但目前依赖于在特定提示条件下评估输出的评估框架是否能可靠地验证真实的记忆修改，仍然存疑。在这项工作中，我们引入了一个简单的诊断框架，使模型在更贴近现实应用环境的上下文学习（ICL）设置下进行判别性自我评估，专门用于审查由记忆修改引起的细微行为差异。这种探测揭示了一种普遍的“表面顺从”（Surface Compliance）现象，即编辑器仅通过模仿目标输出来获得高基准分数，而没有在结构上重写内部信念。此外，我们发现递归修改会累积表示残留，引发认知不稳定，并永久性地降低模型记忆状态的可逆性。这些见解强调了当前编辑范式的风险，并突出了稳健的记忆修改在构建可信、长期可持续的 LLM 系统中的关键作用。代码可在 https://github.com/XiaojieGu/SA-MCQ 获取。

## Abstract
Large Language Models (LLMs) internalize vast world knowledge as parametric memory, yet inevitably inherit the staleness and errors of their source corpora. Consequently, ensuring the reliability and malleability of these internal representations is imperative for trustworthy real-world deployment. Knowledge editing offers a pivotal paradigm for surgically modifying memory without retraining. However, while recent editors demonstrate high success rates on standard benchmarks, it remains questionable whether current evaluation frameworks that rely on assessing output under specific prompting conditions can reliably authenticate genuine memory modification. In this work, we introduce a simple diagnostic framework that subjects models to discriminative self-assessment under in-context learning (ICL) settings that better reflect real-world application environments, specifically designed to scrutinize the subtle behavioral nuances induced by memory modifications. This probing reveals a pervasive phenomenon of Surface Compliance, where editors achieve high benchmark scores by merely mimicking target outputs without structurally overwriting internal beliefs. Moreover, we find that recursive modifications accumulate representational residues, triggering cognitive instability and permanently diminishing the reversibility of the model's memory state. These insights underscore the risks of current editing paradigms and highlight the pivotal role of robust memory modification in building trustworthy, long-term sustainable LLM systems. Code is available at https://github.com/XiaojieGu/SA-MCQ.

*   **核心问题**：现有的评估指标（如 Exact Match）主要关注模型能否生成目标字符串。作者质疑：**模型在输出上“同意”了修改，是否代表其内部“信念”真的被重写了？**
*   **整体含义**：论文提出了“表面顺从”（Surface Compliance）概念，即模型仅在生成层面模仿目标输出，而内部参数化记忆并未真正改变，这种不一致性会导致模型在复杂现实场景中表现脆弱。

### 2. 方法论
*   **核心思想**：从“自由生成”转向“判别性自我评估”，通过在上下文学习（ICL）中引入冲突选项和外部证据，迫使模型暴露其真实的内部记忆状态。
*   **关键技术细节**：
    *   **SA-MCQ 框架**：设计多选题，让模型在“原始答案”、“编辑后答案”及“不确定”选项间做出选择。
    *   **似然边际（Likelihood Margin）**：通过计算编辑前后目标 token 与原始 token 的对数似然差异（$\Delta_{edit}, \Delta_{equiv}, \Delta_{unrel}$），量化概率分布的偏移。
    *   **外部证据刺激**：引入四种上下文证据：
        1.  **参数化证据 (PE)**：支持原始错误记忆。
        2.  **黄金证据 (GE)**：支持编辑目标。
        3.  **无关证据 (IE)**：中性噪声。
        4.  **反事实证据 (CE)**：与原记忆和目标均冲突。
    *   **递归编辑诊断**：设计三轮编辑实验（事实 -> 反事实 -> 回归事实），观察记忆的可逆性和稳定性。

### 3. 实验设计
*   **数据集**：主要使用 **ZsRE**（零样本关系抽取）数据集，因其包含丰富的等价表达和反事实标注。
*   **Benchmark**：对比了传统指标（EM w/ TF, EM w/o TF, LLM-as-judge）与提出的 SA-MCQ。
*   **对比方法**：选取了三种主流编辑器：

*   **通用能力评估**：在 MMLU、GLUE、SST 等标准基准上测试编辑对模型基础能力的影响。

*   **模型规模**：主要针对 LLaMA-3-8B-Instruct 和 Qwen2.5-7B-Instruct。

### 5. 实验数量与充分性
*   **实验规模**：构建了 1K 个高质量测试集，每个样本包含四种证据类型。
*   **充分性分析**：
    *   **多维度验证**：涵盖了概率分布分析、判别性选择、上下文干扰、多轮递归编辑以及通用能力衰减测试。
    *   **鲁棒性检查**：对 SA-MCQ 进行了选项顺序敏感性分析（Position Bias），确保实验结果不是由提示词偏差引起的。

### 6. 主要结论与发现
*   **表面顺从普遍存在**：许多编辑器在 EM 指标上接近 100%，但在 SA-MCQ 判别任务中，模型往往会“反弹”回原始记忆。
*   **记忆的脆弱性**：编辑后的记忆极易受外部上下文影响。即使编辑看似成功，反事实证据也能轻易抑制修改后的行为，使模型陷入认知僵局。
*   **递归编辑的代价**：连续修改同一知识点会累积“表征残留”，导致模型进入亚稳态，永久性降低记忆的可逆性，并引发严重的认知不稳定。
*   **通用能力受损**：高性能编辑器（如 AlphaEdit）在实现强力修改的同时，会导致模型在 MMLU 等通用任务上出现显著的性能坍塌。

### 7. 优点
*   **揭示了评估盲区**：有力地证明了当前知识编辑领域过度依赖生成指标的局限性。
*   **诊断工具高效且深入**：SA-MCQ 框架既能探测深层信念，又具备极高的计算效率。
*   **动态视角**：不仅关注单次编辑，还深入探讨了知识在模型内部演化的长期稳定性。

### 8. 不足与局限
*   **模型规模限制**：实验主要集中在 7B-8B 规模的模型，尚未验证在超大规模模型（如 70B+）中是否存在相同的表面顺从程度。
*   **任务类型单一**：主要基于短格式的 QA 任务，对于长文本生成或复杂逻辑推理中的知识修改探讨较少。
*   **编辑器覆盖面**：未包含内存增强型（Memory-augmented）方法，虽然作者解释是因为这类方法不改变内部参数，但作为对比基准仍具价值。
